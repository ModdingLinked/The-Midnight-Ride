User Interface

In this step, we will install various mods focused on interface visuals and its usability

Creating a Separator in MO2.

  1. Right-click the empty space in the left pane of MO2 and select Create separator.
  2. Name the separator User Interface.

Mod Configuration Menu

Installation instructions:

  • Main Files - Mod Configuration Menu 1.39
Adds an easy way to configure mods through the in-game pause menu.

MCM Settings Manager

Installation instructions:

  • Main Files - MCM Settings Manager
  • MCM Settings Manager - TMR Configuration
Allows you to save, load, transfer, import and export all your MCM settings at once.

Extended Dialogue Interface

Installation instructions:

  • Main Files - Extended Dialogue Interface 1.3.6
Removes the hardcoded 4-option limit on dialogue and adds full engine support for any number of player dialogue options.

Crafting Highlight Fix

Installation instructions:

  • Main Files - Crafting Highlight Fix 1.8.8 (Click the blue name above for a direct link to the correct version)
Replaces the full opacity item highlight during crafting with a much more clear outline.

Workshop Highlight Fix

Installation instructions:

  • Main Files - Workshop Highlight Fix
Changes the workshop object selection from an obscuring shader effect to an outline one, substantially improving the visibility.
