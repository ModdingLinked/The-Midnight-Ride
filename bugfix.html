Buffout 4 NG with PDB support

Fixes engine bugs and adds a crash logger.

Installation instructions

  • Main Files - Buffout4 NG

IMPORTANT:
Mac and Linux users will need to disable the Interior NavCut Fix in Buffout and use the No Multithreading version of the original mod by following these steps:

  • Once the mod has been installed, double-click it in the left pane of MO2
  • In the Text Files tab, select F4SE/Plugins/Buffout4.toml
  • Change the following options:
    • Set InteriorNavCut to false (line 9)
  • Download and install Interior NavCut Fix (No Multithreading)
