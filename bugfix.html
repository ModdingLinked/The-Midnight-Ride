Bug Fixes

In this step, we will install most fundamental bug fixing mods

Creating a Separator in MO2.

  1. Right-click the empty space in the left pane of MO2 and select Create separator.
  2. Name the separator Bug Fixes.

Unofficial Fallout 4 Patch - UFO4P

Installation instructions:

  • Main Files - Unofficial Fallout 4 Patch
Fixes thousands of bugs with both the base game and DLCs.

Ownership Fixes

Installation instructions:

  • Main Files - Ownership Fixes
Fixes incorrect or missing ownership data assigned to junk and containers.

Community Fixes Merged

Installation instructions:

  • Main Files - Community Fixes Merged
A collection of fixes made by the community over the years with open permissions.

The Midnight Ride - Glitchfinder All-In-One

Installation instructions:

  • Main Files - The Midnight Ride - Glitchfinder All-In-One
A small collection of fixes.

Misc Anim Tweaks and Fixes

Installation instructions:

  • Main Files - Misc anim tweaks and fixes
Small animation tweaks and fixes.

Weapon Mod Fixes

Installation instructions:

  • Main Files - Weapon Mod Fixes
  • You will be prompted with a FOMOD installer:
    1. Install
  • Community Fixes Merged - Weapon Mod Fixes Patch
Fixes a number of small and major bugs in vanilla weapon mods.

Buffout 4 NG with PDB support

Installation instructions:

  • Main Files - Buffout4 NG
IMPORTANT:
Mac and Linux users will need to disable the Interior NavCut Fix in Buffout and use the No Multithreading version of the original mod by following these steps:
  • Once the mod has been installed, double-click on it in the left pane of MO2
  • In the Text Files tab, select F4SE/Plugins/Buffout4.toml
  • Change the following options:
    • Set InteriorNavCut to false (line 9)
  • Download and install Interior NavCut Fix (No Multithreading)
Fixes engine bugs and adds a crash logger.

Sprint Stuttering Fix

Installation instructions:

  • Main Files - Sprint Stuttering Fix 1.2 (Click the blue name above for a direct link to the correct version).
Fixes camera stutter when sprinting on uneven surfaces.

General Atomics Galleria Speaking Guard Gutsy Voice Fix

Installation instructions:

  • Main Files - General Atomics Gutsy Guards Voice Fix
  • GA Galleria Gutsy Voice Fix - UFO4P Patch
Fixes the silent Guard Gutsy robots at General Atmoics Galleria.

Companion Jump Fall Pose Fix - F4SE - Garden of Eden SE

Installation instructions:

  • Main Files - Companion Jump Fall Pose Fix (ESP version)
Fixes the "companions stuck in falling pose" bug.

Companion Shoots At Player Fix - F4SE

Installation instructions:

  • Main Files - Companion Shoots At Player Fix
Fixes a bug which causes companions to shoot at the player during or shortly after combat.

Empty Vendor List Bug Fix - F4SE

Installation instructions:

  • Main Files - Empty Vendor List Engine Bug Fix
Fixes an engine bug that can cause vendors' item list to be empty.

Magic Effect and Spell Engine Fixes - F4SE

Installation instructions:

  • Main Files - Magic Effect and Spell Engine Fixes
Fixes a few engine bugs that affect Spells, Inventory Item Enhancements and Perks on NPCs.

Armor Penetration Bug Fix - F4SE

Installation instructions:

  • Main Files - Armor Penetration Bug Fix - F4SE
Prevents incorrect stat overwriting when unequipping or changing weapons.

Outfit ReDress Fix

Installation instructions:

  • Main Files - Outfit ReDress Fix
Engine bug fix to prevent NPCs from defaulting their outfits.

Long Save Bug Fix

Installation instructions:

  • Main Files - GCBugFix Release 0.1
A bug fix for one of the common causes of long save/autosave times.

Reload Fix

Installation instructions:

  • Main Files - Reload Fix
Fixes some reloading issues.

Weapon Behavior Ironsights movement direction lock fixes and tweaks (3rd person)

Installation instructions:

  • Main Files - Ironsights movement direction lock fixes
Fixes movement direction locks, moving in wrong direction and other junk related to input in third person.

Melee Sneak Locomotion Animation Blending Fix (Behavior Edit)

Installation instructions:

  • Main Files - Melee Sneak Locomotion Animation Blending Fix
Adds blending to the 3rd-person melee sneak animations.

Wetness Shader Fix

Installation instructions:

  • Main Files - Wetness Shader Fix
Fixes the exaggerated shine of specific objects while it's raining.

Fixed Protectron Textures

Installation instructions:

  • Main Files - Fixed Protectron Textures (vanilla)
Fixes a wrong texture path for the Medic Protectron and the Robobrain default color.

Fixed Gobo Effects

Installation instructions:

  • Main Files - Fixed Gobo Effects
Fixes artefacts, pixelation and color banding in textures that are used for special light effects.

NPC Drinking Fix

Installation instructions:

  • Main Files - NPC Drinking Fix
Fixes the missing sounds and facial animations of drinking NPCs.

Flutter Flicker Fixer For Foliage

Installation instructions:

  • Main Files - Flutter Flicker Fixer For Foliage
  • You will be prompted with a FOMOD installer:
    1. Archived
    2. PRP 74.6 or later
    3. Install
Fixes the flickering foliage.

Extended Fake Interiors - Commonwealth buildings fixes

Installation instructions:

  • Main Files - Extended Fake Interior
  • Main Files - Extended Fake Interior DLC
  • Optional Files - Extended Fake Interior - CC Enclave Remnants Patch
Fixes various roofs to properly cover you from the rain.

Previsibines Repair Pack

Installation instructions:

  • Main Files - PRP Plugins (1.10.984 Data Files)
  • You will be prompted with a FOMOD installer:
    1. Next
    2. Heavy Incinerator
  • Main Files - PRP Resources
Greatly improves FPS throughout the entire game, including interior cells. Also includes many placement or mesh fixes not possible without regenerating previsibines.

Simple Puddle Reflection Flicker Fix (Water and Blood)

Installation instructions:

  • Main Files - Simple Puddle Reflection Flicker Fix
Fixes water puddles and blood pools flickering and flashing when looking at them from certain angles.

Nvidia-only Section

These last 2 mods should only be installed by users of Nvidia graphics cards. If you are unsure of what graphics card you are using, you can find that information in Task Manager's performance tab.

Weapon Debris Crash Fix

Installation instructions:

  • Main Files - Weapon Debris Crash Fix 1.2 (Click the blue name above for a direct link to the correct version).
This fix is needed only by users of Nvidia GPUs newer than Pascal (10 series).
Fixes crashes with the Weapon Debris setting by disabling debris collision.

Godrays Performance Fix Redux - ESPless

Installation instructions:

  • Main Files - Godrays Performance Fix Redux
This fix is recommended only to Nvidia users because its visual changes do not translate well to AMD hardware, leading to visual problems and a different performance gain.
Optimizes the performance of vanilla godrays.
