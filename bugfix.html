Buffout 4 NG with PDB support

Fixes engine bugs and adds a crash logger.

Installation instructions

  • Main Files - Buffout4 NG

If you plan to play with a controller, you should enable the following setting:

  1. Once the mod has been installed, double-click it in the left pane of MO2.
  2. In the Text Files tab, select F4SE/Plugins/Buffout4.toml
  3. Set InputSwitch (line 32) to true.

Previsibines Repair Pack

Greatly improves FPS throughout the entire game, including interior cells. Also includes many placement or mesh fixes not possible without regenerating previsibines.

Installation instructions

  • Main Files - PRP Plugins NG
  • You will be prompted with a FOMOD installer:
    1. Next
    2. Install
  • Main Files - PRP Resources

Make sure both parts are installed! The mod will do nothing without PRP Plugins, and it'll crash without PRP Resources.
