  2. Name the separator Tweaks.

Unlimited Survival Mode - F4SE

Restores the game features disabled in Survival mode, such as fast travel and normal saves. Fully configurable.

Installation instructions

  • Main Files - Unlimited Survival Mode - F4SE
    1. Once the mod has been installed, double-click it in the left pane of MO2.
    2. In the INI Files tab, select F4SE\MCM\Settings\UnlimitedSurvivalMode.ini
    3. Change iEnableAchievement (line 25) to 0.
    4. Save your changes.
