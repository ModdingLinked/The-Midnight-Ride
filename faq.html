Frequently Asked Questions

Crash before launch

This can be caused by multiple things, so check all of them:
  • Misconfigured Buffout 4, set MaxStdIO to 2048 as instructed in the Buffout 4 section. When done, the setting will look like this: MaxStdIO = 2048.
  • Missing masters, check the big red triangle icon on the top right corner of Mod Organizer 2 and if it says a master is missing, install the mod reported to be missing. Ask for support in the ModdingLinked Discord server if you are unsure on how to do this.
  • Missing AV exclusions, add the required exclusions to your antivirus as instructed in the Adding Exclusions to Defender section.

"Buffout has loaded too late" error

Always run the game with F4SE through Mod Organizer 2 as shown in the picture. If the F4SE option isn't present, restart MO2.

Ultrawide support

Install Ultra Wide Interface (21x9).

Stuck loading a new game

Just like in the High FPS Physics Fix section, open up the configuration file and change NumOfThreadsWhileLoadingNewGame to 2 (line 83). If the issue persists, increase the value until the game manages to load normally.

Missing voice audio or wrong language text

Follow the BethINI section in its entirety, especially setting the language to English.

Door prompts or Locations not in English

You installed the wrong version of PRP's Plugin file. Install the English one from here.

NPCRespawnFix: ERROR: F4SE

  • Follow the BethINI section in its entirety, especially enabling Recommended Tweaks.
  • Always run the game with F4SE through Mod Organizer 2 as shown in the picture. If the F4SE option isn't present, restart MO2.

Invisible or flickering parts of the world

Install the PRP Resources as instructed in the Previs Repair Pack section, then apply the guide's Load Order if you haven't already.

Random crashes

This could also be caused by missing PRP Resources, install them as instructed in the Previs Repair Pack section.

Why aren't the DLC quests starting?

DLC Timing paired with the MCM Settings Manager - TMR Configuration will delay DLCs until you meet the following requirements:
  • Vault-Tec Workshop: level 15 and Reunions completed
  • Automatron: level 20 and Dangerous Minds completed
  • Far Harbor: level 25 and The Glowing Sea completed
  • Nuka World: level 30 and Hunter/Hunted completed
This is fully configurable through the DLC Timing Mod Configuration Menu.

Shouldn't the DLC plugins be cleaned?

Cleaning DLCs is not necessary, and it won't lead to any performance/stability gains.

Merge, Rename, or Replace?

Read the Mod Installation Instructions section.

Application Load Error 5:0000065434 on launch

Run your game directly through Steam once. You may also need to restart Steam and/or Mod Organizer 2. If that still doesn't work, place a copy of your Steam.exe (not a shortcut) into the game's Root folder.

MO2 is reporting conflicts and/or warnings

All file conflicts in the guide are already taken into account and safe to ignore. You can also ignore the overwrite warning. If you see a missing masters warning, create a help thread in the ModdingLinked Discord server with a screenshot of said error and its information.

Why you should disable the High Resolution Texture Pack DLC

The High Resolution Texture Pack provides a very marginal visual increase for a potentially major performance decrease. It is simply not worth the trade-off.

Why are there no big visual mods in the guide?

The headroom you have for visual mods highly depends on hardware, especially VRAM. Including big mods would raise the requirements of the guide too much to still call it vanilla+. The game can also suffer a lot from replacing assets with mods which are usually loose files. So for a proper visual overhaul you will also need to know how to pack files into archives.
