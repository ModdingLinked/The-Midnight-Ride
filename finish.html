Final Steps

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select All Mods -> Create Separator
  2. Name the separator Final Steps

Conflict Resolution FOMOD

Installation instructions:

  1. Download the mod manually begin the installation normally through MO2
  2. You will be prompted with a FOMOD installer, in which you should check all the patch options that apply to what mods you installed
If you installed all the mods in the guide, select every option.

- Resolves all conflicts between the mods in the guide

Load Order.txt

Installation instructions:

  1. Download the mod manually
  2. Once the file has finished downloading, click the button at the top of MO2 and select Open Profile folder
  3. Extract the loadorder.txt from the archive to opened folder and overwrite when prompted
  4. Once the file is extracted, right-click in the left pane of MO2 and select All Mods -> Refresh

After applying the load order and refreshing MO2, Interiors Enhanced - Previs Repair Pack Patch.esp should be at the bottom of your load order (the right pane of MO2). If not, you made a mistake during mod installation. Join the Discord for support.

- Automatically ensures your load order is correct for the guide. This will still work if you didn't install certain mods, but if you added your own mods they will be sorted to the bottom of the load order and need to be manually moved back to their proper position. If you do not know how to check where any mods you have added should go in the load order, it may be best to not add any mods not in the guide.

Ruddy88's Simple Sorter

Installation instructions:

  1. Download and install the Main File - R88SimpleSorter_XML 1.0 normally through MO2
  2. Download the R88SimpleSorter_SCRIPT main file using the Manual Download button
  3. From the downloaded archive, extract the Edit Scripts folder to where you have FO4Edit installed
The Edit Scripts from the archive should align with the Edit Scripts folder already in the FO4Edit folder
  1. Once the folder is extracted, run FO4Edit through MO2
  2. At the Module Selection pop-up, just hold down left shift and click OK
  3. At the ModGroups pop-up, just select OK
  4. Wait for the message Background Loader: finished to appear in the right pane
  5. Right-click anywhere in the right pane and select Apply Script...
  6. In the pop-up, select R88_SimpleSorter from the Script drop-down then select OK
  7. In the Options pop-up, select the following options then select New Patch:
  8. In the next pop-up, make sure every plugin is checked then select OK
  9. In the next pop-up, name the file Ruddy's Simple Sorter then select OK
  10. Click OK again to let the script run (may take a few minutes)
  11. Wait for the message Done: Applying script "R88_SimpleSorter" to appear in the right pane
  12. Press control+s to save the changes, and then exit the program
  13. Once you have exited the program, double-click on the Overwrite mod at the bottom of the left pane of MO2
  14. Drag Ruddy's Simple Sorter.esp from the Overwrite to the Ruddy88's Simple Sorter mod in the left pane of MO2
  15. Check Ruddy's Simple Sorter.esp in the right pane of MO2

- Adds an automatically-compatible item sorting method

Configuring the Mod Configuration Menu

The Mod Configuration Menu can be accessed via the in-game pause menu. Only three mods in the guide have an MCM: Crating Highlight Fix, Extended Dialogue Interface, and Complex Vendors. None of the settings you can change are essential, so you can just adjust them to your personal preference. I personally just leave everything default.

Finishing Notes

Congratulations! You now have modded Fallout 4 set up perfectly and can enjoy a completely stable/smooth experience. You may want to add your own mods on top of the guide, but that is only recommended if you are an experienced modder. If you do not know how to do manual conflict resolution in FO4Edit, how to sort your load order manually, or how to check if a mod is dangerous (like if it breaks pre-combines for example), it is highly recommended to NOT add any of your own mods. And remember, I do not provide support setups that have added mods. The game is very fickle and will break under too much pressure, and many popular mods are broken and will accelerate this process greatly. Here is a list of many popular mods that you should avoid if you are adding any mods. And remember, if you run into any trouble or have any questions, join the Discord and we'll be sure to help you out. DO NOT bother the authors of any of the mods in the guide if you have issues, that's what the Discord is for.
