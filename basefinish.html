Base Finish

Applying the Load Order

The final step is to apply the load order of installed plugins. The guide comes with a premade load order file, which ensures all mods are being loaded in correct order, with all conflicts accounted for.
  1. Download the Load Order from here.
  2. Once it has finished downloading, click the button at the top of MO2 and click Open Profile folder.
  3. Open the downloaded archive and drag loadorder.txt into the folder, overwriting when prompted.
  4. Once the file is extracted, click F5 while you are on MO2 without any menu open.
  5. Right-click in the right pane of MO2 and select Enable all.

If you have not added any mods to the guide, prp.esp should be the last plugin in your load order (the Plugins tab in the right pane of MO2) after applying the load order and refreshing MO2.

If it's not the last plugin after refreshing, make sure that you are sorting the load order by priority by pressing the Priority column on the top.

If that still doesn't work, you can join our Discord server for support.

Running the Game

Due to the way Mod Organizer 2 works, running programs through it is a requirement in order to load mods. This is done in the top-right corner of the manager, where the program to run is chosen. Here, select F4SE.

For the sake of convenience you may also create a shortcut (right below the Run button) so that you won't have to manually open MO2 each time before playing.


You have now completed the basic section of The Midnight Ride. If you don't want to mod your game further, you can now start playing. Have fun!

TMR Extended

Going beyond Vanilla

The next pages are part of the The Midnight Ride Extended portion of the guide, which is not required in order to play. This section includes:

  • UI visual and QoL improvements.
  • More gameplay features and improvements to get the game up to speed with more recent titles.
  • New and restored content, along with improvements to the DLCs' integration.
  • Overhauled balance and game mechanics.
  • Improvements to textures, new animations, visual fixes, and overhauled lighting and weathers.

Note that TMR Extended is completely modular! You can skip any of its mods as long as you do the same for any other mod or patch tied to it.
